Dos2Unix - Convert line-feed plus carriage return (CRLF) to line-feed (LF)

It's a text filter that can convert line ending marker of text from line-feed plus carriage return (CRLF), DOS/Windows-style, to line-feed (LF), Unix-style

Download: Dos2Unix_20100930.zip

Instructions:

  1. Unpack .EXE and .BAT from .ZIP file in a folder
  2. Drag-drop text file onto CRLF2LF.bat to convert it (original file is replaced)

Source code:

Source code in Object Pascal / Delphi (the .DPR file holds the Delphi command-line project code) included
