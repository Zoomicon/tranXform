PPTmerge - merge multiple PowerPoint Presentations into a single .PPT file

Place the .PPT files you want to merge into a folder and drag-drop that folder onto PPTmerge.vbs to launch PowerPoint and have the files merged into a file named after the folder (with _MERGED.ppt appended to the folder name), that is placed inside the drag-dropped folder's parent folder. To enforce a certain merge order, name the files using an alphanumeric (sortable) prefix, for example "1 - Intro.ppt", "2 - Management.ppt", "3 - R&D.ppt" etc.

For the above to work you need PowerPoint installed (had tried previous version with Office97, Office2000, OfficeXP and Office2007 and the latest one with PowerPoint 2010) and Windows Script Host (pre-installed by default at WinXP and Windows Vista, can get it from http://msdn2.microsoft.com/en-us/library/ms950396.aspx)
