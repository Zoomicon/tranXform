XSLer.exe

Given an input XML file and an XSL transformation file, creates the resulting XML, HTML, Text or whatever file the XSL tranformation produces

Download: XSLer_20180204.zip

Syntax: XSLer XMLfile XSLfile outputFile

Note: This tool is deprecated, I suggest you use the XSLTransform utility

Requirements: if you want to rebuild using the source code, you need Borland Delphi. You can download the free "Delphi Personal Edition" from http://www.borland.com/delphi
