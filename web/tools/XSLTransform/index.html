XSLTransform.exe

Given an input XML file and an XSL transformation file, creates the resulting XML, HTML, Text or whatever file the XSL tranformation produces

Download: XSLTransform_Source_20140926.zip

Syntax: XSLTransform xmlFile xslFile outFile

Requirements:

  • To run this executable you need to have the Microsoft .NET runtime installed. You can get it from the Windows Update website
  • To rebuild the VB.net project you can use the Visual Studio.net IDE or the free .NET Framework from http://www.microsoft.com/net (sorry, no NMAKE makefile yet included, you'll have to make one if you want to automate the rebuild [see the .NET Framework samples], or use the VBC.exe compiler from the command-line)
