CleanTextFilter - Replace ASCII text control characters with space chars

It's a text filter that can replace unprintable characters with space chars

Download: CleanTextFilter_20100930.zip

Instructions:

  1. Unpack .EXE and .BAT from .ZIP file in a folder
  2. Double-click .BAT to execute demo
  3. Right-click .BAT and select "Edit" to see how it works (dumping explorer.exe to the command-line and filtering out control chars)

Source code:

Source code in Object Pascal / Delphi (the .DPR file holds the Delphi command-line project code) included
