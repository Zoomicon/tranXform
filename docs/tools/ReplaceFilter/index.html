ReplaceFilter.exe

It's a "DOS" filter that replaces a given string with another while writing text read from the input stream to the output stream

Download: ReplaceFilter_20030101a.zip

Syntax: ReplaceFilter fromToken toToken [ <inputFile [ >outputFile ]]

Requirements: if you want to rebuild using the source code (Delphi), you need the source code for the following units:

  • "ac_strings.pas"
  • "ac_TextFileUtil.pas"
