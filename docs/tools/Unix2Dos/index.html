Unix2Dos - Convert line-feed (LF) to line-feed plus carriage return (CRLF)

It's a text filter that can convert line ending marker of text from line-feed (LF), Unix-style, to line-feed plus carriage return (CRLF), DOS/Windows-style

Download: Unix2Dos_20100930.zip

Instructions:

  1. Unpack .EXE and .BAT from .ZIP file in a folder
  2. Drag-drop text file onto LF2CRLF.bat to convert it (original file is replaced)

Source code:

Source code in Object Pascal / Delphi (the .DPR file holds the Delphi command-line project code) included
