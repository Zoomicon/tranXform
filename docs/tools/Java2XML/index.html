Java2XML

  • JavaClassToXML: Java program to create XML files describing Java classes (using Reflection)
  • JavaResourceBundleToXML: Java program to create XML files with a Java ResourceBundle's contents

Download: Java2XML_20041101.zip

  • Syntax: java JavaClassToXML className > output.xml
  • Syntax: java JavaResourceBundleToXML className > output.xml

Requirements: a Java VM
